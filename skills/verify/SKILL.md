# Verify

Show evidence. Not assertions. Not "should work". Proof.

1. Ask: what will the user actually do? Then do that exact thing — same protocol, same URL, same path.
2. Run the verification command and show the output.
3. Check the project's AGENTS.md or justfile for the right commands.

Evidence: test output, health check responses, curl output, before/after state. Show the command AND its output.
